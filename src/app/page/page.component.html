My Skills

Angular

Angular is a widely used front-end development framework, primarily employed for creating dynamic and interactive web applications.

HTML 5

HTML5 is the latest version of the Hypertext Markup Language, which is essential for building web pages, empowering developers to create richer and more interactive websites.

Javascript

JavaScript is a widely used programming language for web development that plays a fundamental role in creating interactive and dynamic experiences on the internet.

TypeScript

TypeScript is a powerful and increasingly popular programming language that builds upon JavaScript by adding static typing and strong tooling support. It provides developers with the means to write safer and more maintainable code for web and server-side applications.

CSS 3

CSS3, or Cascading Style Sheets Level 3, represents a significant advancement in web styling and design. It introduces a host of new features and capabilities that have transformed the way web developers and designers create visually appealing and responsive websites.

Sass

Sass, or Syntactically Awesome Stylesheets, is a powerful preprocessor scripting language that extends the capabilities of traditional CSS. It provides web developers and designers with a more efficient and maintainable way to write styles for web applications.
